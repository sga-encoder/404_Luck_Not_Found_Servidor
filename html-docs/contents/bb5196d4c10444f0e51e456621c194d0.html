salaJuego

UMLClass
Casino :: Model :: juego :: salaJuegos :: salaJuego

Description

Esta clase crea una sala en el servidor donde los usuarios van a jugar un juego

Specialized Elements

  • JuegoDeCartas
  • KnuckleBones

Attributes

Visibility Name Type Description
protected capacidad Int

la capacidad se refiere a la cantidad maxima de jugadores que pueden entrar a la sala sin quedar en la lista de espera

este valor no puede se menor ni igual a 0
protected capacidadMinimima Int

Hace referencia a la capacidad minima que debe tener la sala para poder jugar al juego

este valor no puede se menor ni igual a 0
protected jugadores Usuario

va a contener lo usuarios que estan jugando en el momento
protected turnoActivo Usuario

turno activo es el jugador que puede tomar accion en ese momento
protected historial JSONArray

El índice X del array Jugadores correspondera al indice X del JSONArray de historial se pontran los resultados de cada ronda entre eltiempo que estubo en la sala
private fechaHoraInicio Date

la hora inicio va hacer hora en la que entras los primeros usuarios a jugar un juego
private fechaHoraFin Date

la hora fin va hacer hora en la que la sala se queda vacia y no hay nadie en cola
private listaEspera Usuario

son los jugadores que estan a la espera de que la sala tenga disponibilidad
protected apuestas Long

El índice X del array Jugadores correspondera al indice X del array de apuestas

aqui contiene todas las apuestas individuales de cada jugador

Visibility Name Description

se eliminara del array jugadores el jugador que salio de la sala, y se guardara su historial de la partida en su Usuario y se borrarar su historial en la sala. y se ejecutara entrarSalaJuego(listaEspera[0], indice del jugador que salio) se comparara el saldo del jugador que acaba de salir de la sala, con el jugadorMayorSaldo, si es mayor se reemplazara
